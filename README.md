# Atelier

## Overview
Redesigned the monolithic API of a retail web portal. Created a microservices oriented architecture to handle web scale traffic.

## Accomplishments
* Optimized a highly scalable microservice to handle up to 5000 requests per second using 4 AWS EC2 T2 micro.
* Improved PostgreSQL query times from 2s to 10ms.
* Utilized stress testing and monitoring tools like K6, Loader.IO, and New Relic to discover possible bottlenecks and limitations from using AWS EC2 T2 micro instances.
* Implemented Nginx caching and load balancing using the least connection load methodology to improve performance.
